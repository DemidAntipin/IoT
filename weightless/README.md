# Описание устройства

- Подключите к плате акселерометр. Используйте пины A4 и A5.
- Подключите к плате светодиод к пину D2.
- Загрузите в устройство [скрипт](weightless.ino)
- Скорректируйте коэффициент `WEIGHTLESS_COEF` под вашу задачу.
